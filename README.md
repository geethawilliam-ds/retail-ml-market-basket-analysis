# retail-ml-market-basket-analysis
Retail analytics project using market basket analysis (FP-Growth) and sentiment analysis for UK retail case study


**Retail Machine Learning & Analytics – UK Case Study
Overview**

This project demonstrates the application of machine learning techniques to a UK-based retail case study involving product bundling strategy and customer feedback analysis.

The objective was to extract actionable business insights using market basket analysis and natural language processing.

**Business Context**

A UK retail shop sought to:

Identify frequently purchased product combinations

Improve cross-selling strategies

Analyse customer feedback sentiment

Support marketing and inventory decisions with data-driven insights

This project applies machine learning techniques to address these objectives.

**Techniques Implemented**


1️⃣ **Market Basket Analysis**

FP-Growth algorithm

Association rule mining

Identification of frequently bought-together products

**2️⃣ Sentiment Analysis**

Text preprocessing

NLTK & TextBlob

Polarity scoring

Customer feedback classification
